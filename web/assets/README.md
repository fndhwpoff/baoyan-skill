# 英文面试练习网页 — 使用说明

## 启动方式

在 Claude Code 中输入：

```
/保研 practice
```

或直接运行：

```bash
python3 tools/web_server.py --port 8765
```

然后在浏览器打开 `http://localhost:8765`

## 功能说明

### 面试流程
1. **准备阶段（30秒）**
   - 屏幕显示英文面试题
   - 你可以在脑中组织答案
   - 30秒后自动进入录制

2. **作答阶段（90秒）**
   - 摄像头录制你的回答
   - 90秒后自动停止
   - 也可以提前点击"结束作答"

3. **回放阶段**
   - 查看刚才录制的视频
   - 发现表达问题或肢体语言缺陷
   - 点击"下一题"继续

### 题目来源
- 优先从题库（`data/questions/index.json`）加载
- 题库通过 `/保研 resume` 生成并保存
- 如题库为空，使用内置示例题目

### 隐私说明
- ✅ 所有录制数据仅存储在浏览器内存中
- ✅ 关闭页面后录制数据自动清除
- ✅ 不会上传到任何服务器
- ✅ 不需要网络连接（除了加载页面）

## 浏览器要求
- Chrome 80+ / Edge 80+ （推荐）
- Firefox 75+
- Safari 14.5+
- 需要摄像头和麦克风

## 快捷键
| 按键 | 功能 |
|------|------|
| Space | 开始/停止录制 |
| → | 下一题 |
| ← | 上一题 |
| Esc | 跳过 |

## 故障排除

### 摄像头不显示
1. 检查浏览器是否阻止了摄像头权限
2. 点击地址栏左侧的锁图标 → 允许摄像头
3. 确保没有其他应用占用摄像头

### 录制按钮不响应
1. 确认已授权麦克风权限
2. 检查浏览器是否为最新版本
3. 尝试重启浏览器

### 题库为空
1. 先运行 `/保研 resume` 生成英文面试题
2. 生成时选择"保存到题库"
3. 确认 `data/questions/index.json` 文件存在
